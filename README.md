
# 📊 Superstore Sales Dashboard | Power BI | SQL|EXCEL

![Dashboard Preview](dashboard.png)

## 🚀 Project Overview

The Superstore Sales Dashboard is an interactive Business Intelligence solution developed in Power BI to analyze retail sales performance, profitability, customer segments, product performance, and regional sales trends.

This dashboard transforms raw transactional data into meaningful business insights, enabling stakeholders to monitor KPIs, identify growth opportunities, and make data-driven decisions.

---

## 🎯 Business Problem

Retail businesses generate large volumes of sales data every day. Without proper analysis, it becomes difficult to:

- Monitor business performance
- Identify profitable products and regions
- Understand customer behavior
- Track sales growth trends
- Improve decision-making

This dashboard was designed to solve these challenges by providing an interactive and visually appealing analytics solution.

## 📋 KPIs Created

### Total Sales

```DAX
Total Sales =
SUM(Superstore[Sales])
```

### Total Profit

```DAX
Total Profit =
SUM(Superstore[Profit])
```

### Total Orders

```DAX
Total Orders =
DISTINCTCOUNT(Superstore[Order ID])
```

### Profit Margin %

```DAX
Profit Margin % =
DIVIDE(
SUM(Superstore[Profit]),
SUM(Superstore[Sales]),
0
) * 100
```

### Average Discount %

```DAX
Average Discount % =
AVERAGE(Superstore[Discount]) * 100
```
